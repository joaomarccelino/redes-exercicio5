Desenvolvimento Frontend Alterado por GIOVANNI

Objetivos:

  • Criar e alterar Website com a linguagem de marcação de hipertexto HTML
  • Desenvolver padrões de apresentação com CSS
  • Criar páginas otimizadas usando JavaScript e bibliotecas interativas
  • Ser capaz de criar sites responsivos para dispositivos móveis utilizando frameworks como apoio ao desenvolvimento

Conteúdo programático

  • HTML: introdução, padrões, tags, formulários e seus componentes.
  • CSS: introdução, histórica, sintaxe geral, seletores, formatação, classes e pseudo classes, propriedades de formatação, comportamento visual, o modelo de caixa e posicionamento.
  • Javascript: introdução, histórico, características, estruturas condicionais, estruturas de repetição, vetores, funções, objetos, manipulação de HTML.
  • Frameworks Front-End: introdução, principais frameworks, características, utilização, web APIs, desenvolvimento responsivo.
